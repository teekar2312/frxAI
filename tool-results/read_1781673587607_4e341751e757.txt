     1‚Üí# FinexFX AI Trading System ‚Äî Worklog
     2‚Üí
     3‚Üí## Project Overview
     4‚ÜíA comprehensive forex live-trading dashboard (Next.js 16 + TypeScript + shadcn/ui + Prisma/SQLite).
     5‚ÜíRepresents the monitoring & operational layer of a Python 3.14 + MT5 (FINEX Indonesia) scalping bot
     6‚Üíwith AI/ML analysis. Pairs: EURUSD, USDJPY, GBPUSD, XAUUSD. Timeframe M5.
     7‚Üí
     8‚ÜíSingle visible route: `/` (dashboard with tabbed sections). Backend = Next.js API routes.
     9‚ÜíLive price feed = mini-service websocket on port 3003.
    10‚Üí
    11‚Üí## Architecture
    12‚Üí- **Frontend**: `/src/app/page.tsx` (single route, client component, tabbed dashboard)
    13‚Üí- **Shared types**: `/src/lib/types.ts`
    14‚Üí- **API base**: `/api/*` (Next.js route handlers, server-side)
    15‚Üí- **DB**: Prisma + SQLite via `@/lib/db`
    16‚Üí- **WS mini-service**: `/mini-services/price-feed/` (port 3003, socket.io)
    17‚Üí- **AI**: z-ai-web-dev-sdk LLM for market analysis (server-side only)
    18‚Üí- **News**: seed + LLM-summarized; web-search skill optional

    26‚Üí## API Contracts (all return JSON)
    27‚Üí
    28‚Üí### Accounts
    29‚Üí- `GET /api/accounts` ‚Üí `{ accounts: Account[] }`
    30‚Üí- `POST /api/accounts` body `{ name, broker, server, login, accountType, currency, leverage, balance }` ‚Üí `{ account }`
    31‚Üí- `PATCH /api/accounts/[id]` body partial ‚Üí `{ account }`
    32‚Üí- `POST /api/accounts/[id]/connect` ‚Üí `{ account, connected: bool }` (toggle MT5 connection)
    33‚Üí- `DELETE /api/accounts/[id]` ‚Üí `{ ok }`
    34‚Üí
    35‚Üí### Trades / Positions
    36‚Üí- `GET /api/trades?status=open|closed&accountId=&symbol=&limit=` ‚Üí `{ trades: Trade[] }`
    37‚Üí- `POST /api/trades` body `{ accountId, symbol, side, lotSize, stopLoss, takeProfit, source, trailingStop, trailingPips }` ‚Üí `{ trade }` (opens position at current sim price)
    38‚Üí- `POST /api/trades/[id]/close` ‚Üí `{ trade }` (closes at current sim price, computes pnl)
    39‚Üí- `PATCH /api/trades/[id]` body `{ stopLoss?, takeProfit?, trailingStop?, trailingPips? }` ‚Üí `{ trade }`
