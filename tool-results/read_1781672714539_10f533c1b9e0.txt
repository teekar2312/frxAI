     1→     1→# FinexFX AI Trading System — Worklog
     2→     2→
     3→     3→## Project Overview
     4→     4→A comprehensive forex live-trading dashboard (Next.js 16 + TypeScript + shadcn/ui + Prisma/SQLite).
     5→     5→Represents the monitoring & operational layer of a Python 3.14 + MT5 (FINEX Indonesia) scalping bot
     6→     6→with AI/ML analysis. Pairs: EURUSD, USDJPY, GBPUSD, XAUUSD. Timeframe M5.
     7→     7→
     8→     8→Single visible route: `/` (dashboard with tabbed sections). Backend = Next.js API routes.
     9→     9→Live price feed = mini-service websocket on port 3003.

    27→    27→
    28→    28→### Accounts
    29→    29→- `GET /api/accounts` → `{ accounts: Account[] }`
